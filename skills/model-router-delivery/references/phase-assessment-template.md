# Phase Assessment Template

## Scope

List the single bounded change and explicit non-goals.

## TDD evidence

Record the initial failing test and the final focused and full-suite results.

## Secondary assessment

Check domain ownership, contract compatibility, security, observability, rollback and documentation accuracy.

## Checklist status

Separate Completed, Partial, Deferred and Blocked. Never convert Partial into Completed without executable evidence.

## Commit

Record the implementation commit hash, then commit this metadata update separately.
