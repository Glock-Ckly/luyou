1. 文本框输入自然语言任务
2. 展示路由决策：`task_type`、`selected_model`、`steps`
3. 展示执行结果 `result` 与子任务拆分
4. `code_patch` 时提示 Cursor Queue id 与 `cursor_cli pop` 指引

## 技术要点

- 后端：`POST /api/route` → `asyncio.run(handle_prompt(body.prompt))`
- 流式：当前 orchestrator 非流式，首版可同步 JSON
- 鉴权：本机 127.0.0.1 可暂不鉴权
