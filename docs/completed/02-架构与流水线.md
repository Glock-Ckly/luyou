# 架构与流水线（已完成）

## 总览图

```
User Prompt
    │
    ▼
┌─────────────────┐
│  Orchestrator   │  src/orchestrator.py
└────────┬────────┘
         │
    ┌────▼────┐
    │Decomposer│  DeepSeek — 是否拆分子任务
    └────┬────┘
         │ 每个子任务
    ┌────▼────┐
    │ L1 关键词│  本地、免费；置信度≥0.7 跳过 L2
    └────┬────┘
         │ 低置信
    ┌────▼────┐
    │ L2 LLM  │  DeepSeek — 11 类 task_type
    └────┬────┘
         │
    ┌────▼────┐
    │ Router  │  task_type→model + 预算四区
    └────┬────┘
         │
    ┌────┴────┬────────────┐
    ▼         ▼            ▼
 Claude     GPT       DeepSeek    cursor_queue
    │         │            │            │
    └────┬────┴────────────┴────────────┘
         ▼
   Response Validator（语法/结构/标记）
         ▼
   Response Aggregator（多子任务合并）
         ▼
   Unified JSON
```
