# 编排器与执行（已完成）

## Orchestrator

文件：`src/orchestrator.py`

主入口：

```python
from orchestrator import handle_prompt
result = await handle_prompt("你的自然语言任务")
```

`MultiModelOrchestrator.handle()` 串联：

1. `_decompose` → `task_decomposer.decompose` + `relay_llm`
2. 对每个子任务 `_classify` → type_hint / L1 / L2
3. `route()` + `_get_budget_ratio()`
4. 执行：LLM 或 `cursor_push`
5. `validate()` — 不合格可重试或升级模型
6. `_aggregate` — 多子任务拼接 + Cursor 任务列表
