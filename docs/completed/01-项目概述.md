# 项目概述（已完成）

## 做什么

接受用户自然语言任务 → 自动分级 → 路由到最合适模型（或 Cursor 队列）→ 返回统一 JSON + 成本信息。

## 模型职责（已落地）

| 任务类型 | 执行目标 | 成本档 |
|----------|----------|--------|
| `architecture` / `system_design` / `deep_reasoning` | Claude（Sonnet/Opus） | 中–高 |
| `implementation` / `debugging` / `refactor` / `uncertain` | GPT（映射 gpt-5.5） | 中 |
| `boilerplate` / `bulk_generation` / `data_processing` | DeepSeek（v4-flash） | 低 |
| `code_patch` / `file_edit` | Cursor Queue（手动） | 低 |

## 统一输出格式

`orchestrator.handle_prompt()` 返回示例字段：

```json
{
  "task_type": "implementation",
  "selected_model": "openai/gpt-5.5",
  "reason": "classified via decompose+classify+route+...",
  "steps": ["decompose", "classify", "route", "execute", "validate", "aggregate"],
  "result": "...",
  "cost_level": "medium",
  "subtask_count": 1,
  "cursor_tasks": 0,
  "total_cost_usd": 0.001
}
```
