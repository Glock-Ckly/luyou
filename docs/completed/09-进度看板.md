# 进度看板（已完成）

## 用途

**监控与进度展示**，不是用户输入界面。

## 启动

```bash
python scripts/dashboard_server.py
```

浏览器：**http://127.0.0.1:1785**

## 文件

| 路径 | 作用 |
|------|------|
| `scripts/dashboard_server.py` | HTTP 服务，端口 1785 |
| `dashboard/index.html` | 深色主题单页 |

## API

`GET /api/status` → JSON：

- `progress` — 里程碑 7/7、100%
- `roadmap` — 路线图状态
- `pipeline` — 架构流水线步骤
- `modules` — 模块表
- `metrics` — smoke/L2/decomposer/e2e/acceptance
- `providers` — 中转站（密钥脱敏）
- `blockers` — 阻塞项（均已 done）
- `budget_ratio` — 实时预算压力
- `routing` — 类型→模型速查

页面每 30 秒自动 `fetch('/api/status')`。

## 已实现增强

- 100% 完成绿色横幅
- 路由映射速查卡片
- `ConnectionAbortedError` 捕获（客户端刷新不断进程）
- `budget_ratio` 线程池 + 3s 超时（防 API 卡死）
