# 文档中心

本目录将项目文档与源码分离，分为两类：

| 目录 | 内容 |
|------|------|
| **[completed/](completed/)** | 已实现、已验收、可运行的能力 |
| **[pending/](pending/)** | 未完成、搁置或待增强的能力 |

## 快速导航

- 已完成总览 → [completed/README.md](completed/README.md)
- 待办总览 → [pending/README.md](pending/README.md)
- 设计原文（仓库根）→ `../multi-llm-router-design.md`
- 运维摘要（仓库根）→ `../STATUS.md`

## 文档与代码关系

- `docs/completed/` 描述**当前 truth**，以 `run_acceptance.py` 通过为准
- `docs/pending/` 描述**缺口与排期**，不代表承诺日期
- 源码、配置、脚本仍在原目录（`src/`、`scripts/`、`config/`），未移动
